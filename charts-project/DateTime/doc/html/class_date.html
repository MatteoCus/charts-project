void Date::check_date ( int  day,
int  month,
int  year 
) [private]

Metodo privato di test date.

Controlla la coerenza delle componenti di una data. In particolare deve risultare:

  • day in [1,32]
  • month in [1,12]
  • year >= 1900
  • month in {4,6,9,11} -> day <= 30
  • month = 2 && is_leap(year) -> day <= 29
  • month = 2 && !is_leap(year) -> day <= 28

altrimenti verra' lanciata un'eccezione
